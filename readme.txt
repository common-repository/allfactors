[AllFactors](https://allfactors.com/) is a no-code all-in-one web marketing analytics tool to drive your marketing & growth.

== Description ==

This plugin connects AllFactors with your WordPress site, giving you immediate access to all of the features.

Your Powerful Features:

* All conversions are measured automatically, no need to set up events
* Video & Audio engagement measured automatically
* Readership score calculated for every page
* Ad performance and attribution for Google Ads, Facebook, Twitter, LinkedIn, X, Bing, and all other platforms.
* SEO and content performance 
* Your own Branded Link Shortener and UTM tracking

== Installation ==

1. Install the official AllFactors plugin by clicking Install Now, and then Activate
2. Under Settings click on AllFactors
3. Paste your AllFactors hostname in the input field and click Save Changes
4. Ask for a raise because you are now a data-driven marketer!

== Frequently Asked Questions ==

= Requirements =
Your website or blog must be using WordPress.org version 5.7 or higher on your server. We recommend the latest version.
The plugin cannot be installed on WordPress.com sites, as they do not allow you to add plugins or Javascript.
You must be a WordPress admin to be able to install plugins for your site. If you are not the admin, you can forward these instructions to the person who manages your WordPress install.
